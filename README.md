A responsive personal portfolio website built with HTML, SCSS (Sass), and modern CSS to showcase my projects, skills, and professional background. This project was created as part of my learning journey and is continuously evolving as I improve my frontend and web development skills.

 Features

Responsive layout (mobile-first approach)

Hero section with call-to-action (Download CV)

About Me section with profile image and professional summary

Skills section with styled skill badges

Projects section showcasing real projects with GitHub links

Contact section with email, LinkedIn, and GitHub

Smooth scrolling navigation

SCSS architecture using partials, mixins, and variables

 Tech Stack:

HTML5 (semantic markup)

SCSS (Sass) with partials and mixins

Flexbox for layout

Responsive design using media queries
