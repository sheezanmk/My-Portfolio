A responsive personal portfolio website built with semantic HTML and modular SCSS to showcase my projects, technical skills, and professional background.

This project serves as a central hub for my work and is designed with a mobile-first approach, clean layout structure, and reusable SCSS architecture.

**Overview**

The portfolio was built to:

Present the projects built over time

Highlight technical skills and professional experience

Provide recruiters and hiring managers with easy access to my GitHub and LinkedIn

Demonstrate clean layout structure and responsive design principles

**Features**

Responsive layout (mobile-first)

Hero section with call-to-action (Download CV)

About section with professional summary

Skills section with styled badge components

Project showcase cards with GitHub links

Contact section with direct navigation links

Smooth scroll navigation

Modular SCSS architecture using partials and mixins

**Tech Stack**

HTML5 (semantic structure)

SCSS (Sass) with partials and mixins

Flexbox for layout

Media queries for responsive design

Font Awesome icons

**Architecture & Approach**

The project follows a modular SCSS structure:

Separation of concerns using partial files

Reusable mixins for responsive breakpoints

Consistent spacing and layout scale

Component-based section styling (About, Skills, Projects, Contact)

The layout was built mobile-first and progressively enhanced for desktop screens using media queries.

**Setup**

To run locally:

git clone <repo-url>

Open index.html in your browser.
